# Aura Verde

**Aura Verde** — интернет магазин экологически чистых товаров.

## Запуск

По ссылке https://annabarba-2005.github.io/aura-verde/ или локально через `Live Server`

## Технологии

* **HTML5** — семантическая разметка
* **CSS3** — переменные и блочная структура стилей
* **JavaScript** — без библиотек и сборки
* **Статика** — не требует backend или сборщика
